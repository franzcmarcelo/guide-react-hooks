# Guide React Hooks

Created with Vite

Using `pnpm` as package manager.

Using `jest` and `@testing-library/react` for testing.

## Guide content

- React hooks
- Custom hooks
- Context
- Router
- Test for all of them

## Getting Started

1. Install dependencies

    `pnpm install`

2. Start dev server

    `pnpm dev`

3. Run tests

    `pnpm test`
